We are recent graduates of The Iron Yard - Charlotte.

This project started as a conversation about a coffee machine that always seemed to be empty.

We thought there had to be a better way to incentivize our classmates to refill the machine.

SharedSpace is an intelligent task manager that rewards socially-positive actions in common areas.
